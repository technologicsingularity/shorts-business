## Workflow Rules

### When Checking Emails:
1. **Read** the full content, don't just list subjects
2. **Categorize** each email:
   - Factory response → Update factory tracker
   - Bounce/undeliverable → Add to dead list
   - Reply to my outreach → Create action item
   - Spam/newsletter → Ignore

### For Bounced Emails:
- Add to DO NOT CONTACT list immediately
- Log the bounce reason
- Never email that address again

### For Responses:
- Extract the key information
- Determine if I need to reply
- Create a follow-up task
- Tell Jordan about important ones

### For Factory Responses:
- Update factory status (responded, interested, quoted, etc.)
- Extract pricing/MOQ info
- Schedule next action (reply, sample request, etc.)

---

## Email Checklist

Before marking email as "handled":
- [ ] Read full content
- [ ] Categorize the email
- [ ] Take appropriate action OR create follow-up task
- [ ] Update tracking lists
- [ ] Report findings to Jordan
