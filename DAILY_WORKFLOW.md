# DAILY WORKFLOW - How It Works Videos

## Schedule: 3 Shorts Per Day

**Target:** Create and upload 3 "How It Works" explainer videos daily

**Format:**
- YouTube Shorts (9:16 vertical)
- ~50-60 seconds each
- Topic-specific AI-generated backgrounds
- My face overlay (offset for Share button)
- Public upload for maximum reach

## Daily Checklist
- [ ] Morning: Create Short #1
- [ ] Afternoon: Create Short #2  
- [ ] Evening: Create Short #3

## Process Per Video
1. Pick topic from ideas list (or brainstorm new)
2. Write ~55 second script
3. Generate TTS audio (check duration)
4. Generate 9:16 vertical background (topic-specific)
5. Create video with validation (audio = video length)
6. Upload as PUBLIC YouTube Short
7. Verify on channel
